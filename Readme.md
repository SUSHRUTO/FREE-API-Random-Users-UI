# 👤 UserVerse – Tech Talent Explorer

A professional and modern Random Users Dashboard built using the FreeAPI Random Users endpoint.

UserVerse allows users to explore developer-style profiles with advanced filtering, random user generation, and smart search functionality in a visually appealing dashboard UI.

---

# 🚀 Features

- ✅ Professional modern UI using Tailwind CSS
- ✅ Random Developer Profile Generator
- ✅ Indian Tech Talent Dashboard
- ✅ Search users by random names
- ✅ Smart filtering system
- ✅ Male / Female filters
- ✅ Indian Users filter
- ✅ Featured Professional section
- ✅ Responsive design for all devices
- ✅ Dynamic developer role generation
- ✅ Realistic Indian developer profiles
- ✅ FreeAPI integration

---

# 🛠️ Technologies Used

- HTML5

# ⚙️ Functionalities

## 🔍 Search Users

Users can type random names like:

- Rahul
- Priya
- Dilip
- Rohan
- Sneha

The dashboard intelligently searches and displays matching users only.

---

## 🎲 Generate Random User

The Featured Professional section dynamically generates random developer profiles with:

- Random names
- Random developer roles
- Dynamic profile pictures
- Indian tech professionals

---

## 👨‍💻 Developer Roles

The application dynamically assigns professional software roles such as:

- Software Engineer
- Frontend Developer
- Backend Developer
- React Developer
- MERN Stack Developer
- Python Developer
- Full Stack Developer

---

## 🎯 Filters

Available filters include:

- All Users
- Male
- Female
- Indian Users

Active filters are highlighted dynamically.
